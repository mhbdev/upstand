# @upstand/config (`packages/config`)

The `@upstand/config` package provides shared tooling and configuration files used across all applications and workspace packages in the Upstand monorepo.

## Shared Configurations

- `tsconfig.json`: Base TypeScript compiler options (`strict: true`, target ES2022, bundler module resolution).
- `biome.json`: Shared Biome linter and formatter settings.
- `tailwind.config.ts`: TailwindCSS theme tokens, CSS variable mappings, and custom utility presets.
