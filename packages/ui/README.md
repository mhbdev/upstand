# @upstand/ui (`packages/ui`)

The `@upstand/ui` package is the shared UI component library for Upstand, built with React 19, TailwindCSS v4, shadcn/ui primitives, Radix UI, Lucide React icons, and CodeMirror.

## Components Included

- **Primitives**: `Button`, `Input`, `Dialog`, `AlertDialog`, `Tabs`, `Card`, `Badge`, `DropdownMenu`, `Switch`, `Toast`.
- **Code Mirror Editor**: Unified syntax-highlighted code editor component (`CodeBlock`, `Editor`) with YAML, JSON, Caddyfile, and shell language support.
- **Interactive Modals**: `ConfirmActionDialog` for 2FA-verified destructive action confirmations.
- **Icon Set**: Standardized Lucide React icon exports.

## Usage

```tsx
import { Button, Dialog, ConfirmActionDialog } from "@upstand/ui";
```
