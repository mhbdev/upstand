# Code of Conduct

We want Upstand to be a welcoming, technically rigorous community. Participants must be respectful, constructive, and inclusive regardless of experience, identity, background, or viewpoint.

Unacceptable behavior includes harassment, discrimination, threats, personal attacks, sexualized conduct, doxxing, deliberate disruption, and publishing private information without consent. Critique code and decisions, not people. Assume good intent while addressing impact clearly.

Maintainers may remove comments, close discussions, or restrict participation when behavior violates this standard. Report concerns privately through the channel in [SECURITY.md](SECURITY.md) or to the repository maintainers. Retaliation against a good-faith report is not acceptable.
